# 📝 Text Toolbox

テキスト関連の便利ツールを提供するPWA（Progressive Web App）です。

## 🌟 機能

### 1. 文字コード表示
入力されたテキストの各文字について、以下の情報を表示します：
- Unicode（U+表記）
- 10進数表記
- 16進数表記
- 8進数表記
- 2進数表記
- UTF-16表記

### 2. 正規表現チェッカー
- 正規表現パターンのテスト
- マッチ箇所のハイライト表示
- マッチ位置とグループの詳細表示
- グローバル、大文字小文字区別なし、複数行モードのサポート

## 🛠️ ローカルでの実行

```bash
# リポジトリをクローン
git clone https://github.com/temeteke/text-toolbox.git
cd text-toolbox

# ローカルサーバーで起動（例：Python）
python3 -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```
