# 📝 Text Toolbox

テキスト関連の便利ツールを提供するPWA（Progressive Web App）です。

## 🌟 機能

### 1. 🔢 文字解析
入力されたテキストの各文字について、以下の情報を表示します：
- Unicode（U+表記）
- 10進数、16進数、8進数、2進数表記
- UTF-16、UTF-8表記
- HTML実体参照、CSS、JavaScriptエスケープ
- Unicodeブロック情報
- 各種フォントでの表示

### 2. 🔍 正規表現チェッカー
- 正規表現パターンのテスト
- マッチ箇所のハイライト表示
- マッチ位置とグループの詳細表示
- グローバル、大文字小文字区別なし、複数行モードのサポート

### 3. 🔄 エンコーディング変換
- **Base64**: エンコード/デコード
- **URL**: URLエンコード/デコード
- **HTML**: HTMLエンティティエンコード
- **Punycode**: 国際化ドメイン用エンコード/デコード

### 4. 📊 テキスト統計・解析
- 総文字数、文字数（空白除く）
- 行数、単語数
- バイト数、UTF-8バイト数
- 文字種別の内訳（ひらがな、カタカナ、漢字、英字、数字、記号）

### 5. ✨ テキスト整形
- **JSON**: 整形、最小化
- **行操作**: ソート（昇順/降順）、重複行削除、空行削除
- **空白処理**: 各行のtrim、タブ⇔スペース変換

### 6. 🔤 文字列変換
- **大文字/小文字**: 相互変換
- **全角/半角**: 相互変換
- **かな変換**: ひらがな⇔カタカナ
- **ケース変換**: キャメルケース、スネークケース、ケバブケース、パスカルケース

### 7. 🔐 ハッシュ生成
- MD5（簡易実装）
- SHA-1
- SHA-256
- SHA-512

### 8. 📝 差分比較
- 2つのテキストの行単位での差分表示
- 追加、削除、変更なしを色分けして表示

### 9. 🎲 ランダム生成
- **パスワード**: 文字種選択可能（大文字、小文字、数字、記号）
- **UUID**: UUID v4生成
- **Lorem Ipsum**: ダミーテキスト生成

## 🛠️ ローカルでの実行

```bash
# リポジトリをクローン
git clone https://github.com/temeteke/text-toolbox.git
cd text-toolbox

# ローカルサーバーで起動（例：Python）
python3 -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```
